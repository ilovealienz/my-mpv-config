# mpv config

Personal mpv configuration for Linux (Wayland/Vulkan) and Windows.

`mpv.conf` uses Trebuchet MS for subtitles and OSD. To use it, create a `fonts/` folder in your mpv config directory and place `trebuc.ttf` in it.
