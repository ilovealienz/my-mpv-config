Shift+Tab
 Skip to next silence gap. Intended for skipping show intros — most have a silence gap at the end before the episode starts. Detects silence (threshold −30dB, min 0.5s), speeds to 100× and blackouts video. Jumps to the next audio. Press again mid-skip to cancel and return to where skip began; once it has already jumped it just stops at the new position.
Media Next
 Next track, or restart playlist if at end / only 1 item. Unpauses after jump. Handles the edge case where mpv would otherwise stop at EOF with a single-file playlist.

Subtitles

sub-scale.lua · input.conf
Alt+G
 Increase subtitle font scale. Smart increment stepping: tries 0.025, falls back to 0.5, then 1.0 if mpv doesn't accept smaller steps.
Alt+F
 Decrease subtitle font scale.
R
 Move subtitles up (sub-pos −0.25). reassigned Default: cycle sub track. Now moves subtitle position. Tracks a float internally, rounds to integer for mpv.
T
 Move subtitles down (sub-pos +0.25). reassigned Default: show progress bar. Now moves subtitle position.
Alt+Ctrl+U
 Cycle subtitle blur. cycles 0.0 → 0.5 → 1.0 → 1.5 → 2.0. Useful for softening hard-edged bitmap subs.
Alt+U
 Toggle subtitle monochrome. toggle Strips colour styling, forces white text.
Alt+Shift+U
 Cycle .ass subtitle override mode. cycles strip → force → yes → no. Controls how styled .ass subtitles are rendered; "strip" removes all styling, "force" overrides with mpv defaults.

Defaults (mpv.conf): font=Trebuchet MS · sub-gauss=0.8 · sub-pos=103.5 · sub-scale=1.0 · sub-color=#f7f7f7 · sub-border-color=#121212 · sub-border-size=3.6 · sub-shadow-color=#404040

Upscaling Shaders

shaders.lua
Ctrl+1
 Anime4K HQ 3 presets Mode A+A (Restore VL + Upscale VL + Restore M + Upscale M) → Mode B+B (Restore Soft) → Mode C+A (Upscale Denoise + Restore M) → off. Full CNN restore + upscale pipeline, heaviest on GPU.
Ctrl+2
 Anime4K 3 presets Mode A → B → C → off. Lighter than HQ — same upscale stages but no double-pass restore.
Ctrl+3
 Ani4Kv2 2 presets Ani4Kv2 ArtCNN C4F32 → CMP variant → off. v2 generation ArtCNN upscaler; CMP is a smaller/faster build.
Ctrl+4
 AniSD 2 presets AniSD ArtCNN C4F32 i4 → CMP → off. Tuned for SD source material; 4-pass inference.
Ctrl+5
 FSRCNNX 2 presets x2 8-0-4-1 (faster) → x2 16-0-4-1 (higher quality) → off. Works well on live-action too, not just anime.
Ctrl+0
 Clear all GLSL shaders immediately. Resets internal state so the next Ctrl+1–5 press starts from preset 1 again. Each key also clears any other active family before loading its own.

Audio

input.conf
Alt+Shift+M
 Toggle stereo → mono mix. toggle lavfi pan filter: c0=c0+c1. Mixes both channels into one. Useful for headphones with centred dialogue.
F1
 Cycle Dynamic Audio Normalisation. cycles dynaudnorm g=7 → g=5 → g=3 → off. Higher g = more aggressive smoothing over time.
F2
 Toggle EBU R128 loudness normalisation. toggle loudnorm I=−16 TP=−1.5 LRA=11. Targets broadcast loudness standard.
F3
 Toggle dynaudnorm compressor + soft clip. toggle dynaudnorm f=75:g=25:n=0:p=0.58 drives the compression; volume softclip=yes prevents hard clipping on peaks. More aggressive than F1.

Audio language priority: Japanese Stereo → Japanese → English Stereo → English (set in mpv.conf alang)

Clipboard

SmartCopyPaste.lua
Ctrl+V
 Paste and load file or URL from clipboard. Uses wl-paste (Wayland). Supports file paths, http/https URLs, magnet links. If the clipboard contains a timestamp in the format path |time=N, seeks to N−0.65s for a brief preview before the exact position.

Ctrl+C copy is commented out in SmartCopyPaste.lua — only paste is active. To enable, uncomment the ctrl+c binding at the bottom of the script. The copy format is path |time=N.

Background Scripts

no keybinds — automatic
titles.lua Sets the window title bar on every file load, updating every 3 seconds. Video (mp4, mkv, avi, flv, mov): (01/12) Filename [Chapter] - mpv
Audio (mp3, flac, m4a, wav, wma, aac, dsd, dsf, mqa): (01/12) Artist - 03 - Title [Album, Year, codec kbps, ext] - mpv. For FLAC, bitrate is replaced with bit depth / sample rate (e.g. 24bit / 96kHz). m4a omits the file extension from the bracket. Also resets audio filters (af) on audio file load.
auto-image-playlist.lua When an image is opened, auto-adds all other images in the directory to the playlist alphabetically. Supported: jpg, jpeg, png, bmp, gif, webp, tiff, tif, apng. Animated formats (gif, apng) loop the individual file; static images do not. Playlist itself never loops. Only runs if the playlist has one item — won't re-add if images are already loaded.
sub_path.lua Expands subtitle search paths on every file load. Adds Sub/, Subs/, Subtitles/ as search directories. Also adds Subs/<filename-no-ext> for per-file subtitle folders. Deduplicates paths case-insensitively.
sub-select.lua Auto-selects the best subtitle track based on active audio language. Rules from sub-select.json (in order):
1. Japanese audio → English subs, blacklisting tracks named "sign" or "s&s"
2. English/unknown audio → English/undefined, external subs only
3. English/unknown audio → English/undefined, forced subs only
4. English/unknown audio → English/undefined (any sub)
5. Fallback: undefined language subs
Runs synchronously at preload. Detects and corrects wrong audio track predictions.

Overridden & Disabled Defaults

input.conf · scripts

Keys marked here differ from mpv defaults.

R
 Reassigned: move subtitles up. Default was: cycle subtitle track forward.
T
 Reassigned: move subtitles down. Default was: show playback progress bar.
F
 Ignored (disabled). Default was: toggle fullscreen. Disabled to avoid accidental fullscreen from subtitle size keys.
G
 Ignored (disabled). Default was: cycle subtitle track (some builds). Disabled, subtitle scale handled by sub-scale.lua on Alt+G/F.
E
 Ignored (disabled). Default was: cycle edition (set of chapters).
Enter
 Ignored (disabled). Default was: skip to next playlist item. Disabled to prevent accidental skips.
Ctrl+O
 Disabled (commented out in file-browser/main.lua). Default file-browser binding was: open browser without toggle. Removed to keep Ctrl+B as the only browser key.
Scroll on OSC audio/sub
 Scroll direction reversed on OSC audio and subtitle track buttons. Configured in osc.conf via audio_track_wheel_down/up_command and sub_track_wheel_down/up_command.
af on audio load
 Audio filters reset automatically on audio file load. titles.lua sets af="" on each audio file to prevent filters from carrying over between tracks.
